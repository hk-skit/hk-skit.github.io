Random Utils

Random functions to get random stuff.

Posted by Hitesh Kumar on March 18, 2017

A programmer often need a random number, integer, color, alphanumeric string etc. in his/her day to day life. After googling for sometime, I found that there's no place where I can find all these functions together. So I decided to write a post to put them together.

getRandomNumber(a, b): An overloaded function which returns a random number based on the arguments passed to it. 

/**
 * Returns a random number in range of [a, b]. If only a is passed then
 * returned number is less than a. If no arguments then between(0, 1).
 */
function getRandomNumber(a, b) {
    let n = Math.random();

    if (arguments.length === 1) {
        n *= +a;
    } else if (arguments.length === 2) {
        n *= Math.abs(a - b);
        n += Math.min(a, b);
    }
    return n;
}

getRandomInteger(a, b): An overloaded function which returns a random integer based on the arguments passed to it. 

/**
 * Returns random integer.
 */
Earlier ES6:
function getRandomInteger() {
    return ~~getRandomNumber.apply(null, arguments);
}

ES6:
function getRandomInteger() {
    return ~~getRandomNumber(...arguments);
}

Note: We are using ~~(double tilde) operator to get integer part of generated number. You can read about double tilde here or here. We will not dive into double tilde here cause it's worth writing a whole new post.

getRandomColor(): Returns a random hex code as string. 

/**
 * Returns a random hex code as string.
 */
function getRandomColor() {
    var color = getRandomNumber() * 0xFFFFFF; // random number less than max hex value.
    color = ~~color; // integer part
    color = color.toString(16);     // equivalent hex code.
    while (color.length < 6) {   // pad zero to make length 0.
        color = '0' + color;    // pad zero
    }
    return '#' + color;
}

getRandomString(length): Returns a random alphanumeric of desired length. 

                        /**
 * Returns a random string of passed length. If no length is passed then 
 * string of length 6 is returned.
 */
function getRandomString(length) {
    var str = '';
    var ALPHANUMERIC = 'ABCEFGHIJKLMNOPQRSTUVWXYZabcefghijklmnopqrstuvwxyz0123456789';
    length = length || 6; //default length;
    while (str.length < length) {
        str += ALPHANUMERIC[getRandomInteger(ALPHANUMERIC.length)];
    }
    return str;
}
